## Travel-旅遊網 ✈️
查詢旅遊路線，提供加入購物車、結帳功能。

## 特點 🔆
* 使用React，建立SPA，包含首頁、旅遊資訊、註冊登錄頁、購物車、結帳功能。
* 使用Redux toolkit管理API。
* 搭配CSS Module完成樣式。
* 使用react-i18next切換首頁語言。
* redux-persist管理jwt登入狀態。

## 預覽畫面 📝
### 首頁
* 搜尋旅遊路線。
* 提供推薦行程，點即可查看詳請。
* 提供切換語言功能。
* 提供註冊、登錄。
* 已登錄則顯示購物車數量。
